# CamPersist

A centralized repository documenting various solutions for persisting camera recordings to storage systems like NAS or cloud object storage. This repository serves as a knowledge base for different camera setups and their corresponding storage solutions.

## Overview

This repository contains documentation and implementation details for various camera-to-storage solutions. Each camera setup is organized in its own directory with detailed documentation about the specific implementation.

## Camera Solutions

Currently documented solutions:

- [Xiaomi C700](./xiaomi-c700/) - Camera setup at parents' home with storage persistence solution

## Contributing

To add a new camera solution:

1. Create a new directory for your camera setup
2. Add a detailed README.md describing:
   - Camera specifications
   - Storage solution details
   - Implementation steps
   - Configuration details
   - Any scripts or tools used
